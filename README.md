# Snake

A classic Snake game that runs in your terminal, built with Python's `curses` library.

## Requirements

- Python 3.6+
- No external dependencies

## How to Run

```bash
python3 snake.py
```

## Controls

| Key              | Action     |
|------------------|------------|
| Arrow keys / WASD | Move       |
| Q                | Quit       |
| R                | Restart (after game over) |

## Gameplay

- Eat the red `*` to grow your snake and earn points (+10 each).
- The snake speeds up as your score increases.
- Avoid hitting the walls or your own tail.
